zk-snark算法的核心步骤

The main procedure for using the zk-SNARK algorithm typically consists of the following steps:
Setup : The prover and verifier agree on a set of public parameters and a common reference string, which will be used to define the underlying mathematical structures needed for the proof.
Proving : The prover generates a proof, which consists of a set of mathematical statements that demonstrate that they know the required information, without revealing any actual information about the information itself.
Verification : The verifier receives the proof and checks it against the common reference string and the public parameters to ensure that it is valid. If the proof is valid, the verifier can be confident that the prover knows the required information, without learning any information about the actual information itself.
